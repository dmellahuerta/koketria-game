# Front (Vite)

## Rutas

- `/` juego principal.
- `/map` editor base de terreno custom.
- `/config.html` herramienta de hitbox solo para desarrollo (`DEV`).

## Desarrollo

```bash
npm run dev
```

## Build

```bash
npm run build
```

El build publica `index.html` y `map.html` (multipage). La pûÀgina `config.html` no se publica en `prod`.
